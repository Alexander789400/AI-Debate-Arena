
# AI Debate Arena

AI Debate Arena is an interactive web application where multiple AI agents debate against each other on a given topic, and a Judge AI decides the winner. The system simulates structured debates with Pro Agent, Con Agent, and Judge Agent, making it useful for exploring arguments from multiple perspectives.

## Features :

AI vs AI Debate System

Judge AI evaluates the arguments

Optional web search for real-time information

## Five-agent architecture

Pro Agent

Planner Agent

Con Agent

Judge Agent

Result Agent
